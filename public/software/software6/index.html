BayesMallowsSMC2: Nested Sequential Monte Carlo for the Bayesian Mallows Model

May 2026 Øystein Sørensen

This R package provides nested sequential Monte Carlo algorithms for performing sequential inference in the Bayesian Mallows model.

R package on CRAN

Description

BayesMallowsSMC2 implements SMC2 (Sequential Monte Carlo Squared) algorithms for sequential inference in the Bayesian Mallows model. The package supports sequentially arriving rank and preference data, including complete rankings, partial rankings, and pairwise comparisons.
